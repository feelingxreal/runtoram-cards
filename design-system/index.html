01 · Логотип

Знак

Логотип-стрелка со свечением. Циклится по трём фирменным акцентам — оранж → лайм → индиго. Есть 3D-стеклянная версия для обложек и hero-блоков.

Векторный знак · перекрашивается в любой цвет
Ink
Orange
Lime
Indigo

Чистый SVG — масштабируется без потери качества, цвет через background: var(--rtr-…).

Лента-паттерн из знаков — фоновый / разделительный элемент.
