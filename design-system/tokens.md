# RunToRam — Design System

Дизайн-система сервиса обмена электронных валют RunToRam. Извлечена с боевого сайта
`info.runtoram.com` и сведена с фирменными ассетами (логотип, 3D-рендеры). Палитра и
шрифт сайта совпадают с Instagram-карточками — бренд единый.

> Живой styleguide: открой `index.html` в браузере. Его же можно импортировать в Figma
> по слоям через плагин **html.to.design** (вставить URL страницы → Import).

## 2. Типографика

- **Дисплей / заголовки:** `Roboto Condensed` — веса 700 / 800 / 900, плотный трекинг
  (`-0.02…-0.04em`). Крупный, сжатый, уверенный.
- **Основной текст:** `Roboto` — 400 / 500 / 700.

| Роль | Размер | Шрифт / вес |
|---|---|---|
| Display | 65–83px | Condensed 800, трекинг −0.04em, line-height ~0.9 |
| H1 | 36px | Condensed 800 |
| H2 | 28px | Condensed 800 |
| H3 | 22px | Condensed 700 |
| Lead | 18px | Roboto 500 |
| Body | 16px | Roboto 400 |
| Small | 14px | Roboto 400 |
| Caption | 12px | Condensed 700, UPPERCASE, разрядка +0.08em |

---

## 3. Сигнатурные приёмы

1. **Заголовки с заливкой** — ключевые слова в плашке-маркере с инверсным текстом
   (`We [make] money [moves]`). Плашка: фон бренд-цвет, текст контрастный, `radius 6px`.
2. **Дисплей + контур** — крупные слова сплошным цветом + контурный (outline) вариант.
   **Рецепт контура (Figma):** Roboto Condensed **Bold** · Fill: none · Stroke **Outside 2–3px**
   (`#1C1C1C` или бренд-цвет) · Letter-spacing 0. Обводка снаружи буквы → чистый контур, не
   залезает на глиф. В CSS — приближение через `-webkit-text-stroke` (рисуется по центру);
   для пиксель-точности используй Outside-обводку в Figma.
3. **Мотив скобок `{ }`** — обрамляют цифры/валюты в типографике; есть 3D-стеклянная версия.

---

## 4. Радиусы и отступы

- Инпуты `6px` · мелкие блоки `8px` · карточки `20px` · кнопки `30px` · теги/круги `999px`.
- Шкала отступов (px): `4 · 8 · 12 · 16 · 24 · 32 · 48 · 64 · 80`.

---

## 5. Компоненты

- **Кнопки:** pill (`radius 30px`), Condensed 700. Варианты: оранж/белый,
  ink/белый, индиго/белый, лайм/ink, контурная (ink-бордер).
- **Теги/бейджи:** pill, Condensed 700 UPPERCASE 12px, бренд-цвета.
- **Карточки:** `radius 20px`. Светлая (крем), тёмная (ink), акцентная (индиго).
- **Инпуты:** `radius 6px`, бордер `#D0D5DD`, Roboto 15–16px.

## 7. Тон бренда

Финтех, которому доверяют: уверенный, прямой, «деньги в движении». Крупная сжатая
типографика, насыщенные акценты, минимум воды. Спокойный текст — смелые акценты.
